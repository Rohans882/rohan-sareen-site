Shape Matching Game - 10 minute game challenge

Source Code Download


 This is an extremely simple game I rushed myself to do in 10 minutes. The objective here is to match the shape of the enemy shapes (in red) by clicking Q, W, or E. Each of those keys respond to a specific shape. I tried making this game as fast as possible so I used a random image I was working on as the background, and quickly wrote all of the directions on it. I drew the shapes extremely fast and split them up using a sprite sheet. The following script shoots out the shape pertaining to the key pressed down!

void Update ()
{
 if (Input.GetKeyDown(KeyCode.Q))
{
 Instantiate(projectiles[0], transform.position, Quaternion.identity);
}
 else if (Input.GetKeyDown(KeyCode.W))
{
 Instantiate(projectiles[1], transform.position, Quaternion.identity);
}
 else if (Input.GetKeyDown(KeyCode.E))
{
 Instantiate(projectiles[2], transform.position, Quaternion.identity);
}

JavaScript Dice Game

Source Code Download


 The point of this game is to have players roll dices. The user can choose to roll the dice as many times they want but if they roll a 1 on either of the 2 dice, their score is set to 0 and it is the other player's turn. The point is to roll maybe once or twice and then click the Hold option to keep the rolls. I made it so that a user can set what number they can play too by typing a number in the Final Score box, or else it will be 100 by default. If the rules explained are hard to understand, maybe this small script will help!
